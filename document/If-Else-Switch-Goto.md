# If-Else / Switch / Goto

C++ มีเครื่องมือที่ช่วยให้คุณควบคุมพฤติกรรมของโปรแกรม (ลำดับการทำงานของตรรกะ หรือ Logic Flows) ตามการโต้ตอบของผู้ใช้ ในบทนี้เราจะพูดถึง `if-else`, `switch` และ `goto` ซึ่งเป็นสามวิธีพื้นฐานในการกำหนดทิศทางการทำงานของโค้ด

## if-else
ใช้ `if-else` เมื่อคุณต้องการให้โปรแกรมเลือกระหว่างสองทางเลือกที่เป็นไปได้

```cpp
#include <iostream>

int main() {
    int age = 18;

    if (age >= 18) {
        std::cout << "You can vote!" << std::endl;
    } else {
        std::cout << "Too young to vote." << std::endl;
    }

    return 0;
}
```
**คำอธิบาย:** โปรแกรมจะตรวจสอบเงื่อนไขภายใน `if` ถ้าเป็นจริง (true) บล็อกแรกจะทำงาน แต่ถ้าเป็นเท็จ (false) บล็อกภายใต้ `else` จะทำงานแทน

## switch
ใช้ `switch` เมื่อคุณมีผลลัพธ์ที่เป็นไปได้หลายแบบสำหรับตัวแปรเพียงตัวเดียว ซึ่งมักจะดูสะอาดและอ่านง่ายกว่าการใช้ `if-else` ซ้อนกันหลาย ๆ ชั้น

```cpp
#include <iostream>

int main() {
    int day = 3;

    switch (day) {
        case 1: std::cout << "Monday"; break;
        case 2: std::cout << "Tuesday"; break;
        case 3: std::cout << "Wednesday"; break;
        default: std::cout << "Invalid day";
    }

    return 0;
}
```
**คำอธิบาย:**
1. ค่าของตัวแปร `day` จะถูกนำไปเปรียบเทียบกับแต่ละ `case`
2. เมื่อเจอกรณีที่ค่าตรงกัน โค้ดในบล็อกนั้นจะเริ่มทำงาน
3. `break` ใช้สำหรับหยุดการทำงานและออกจาก switch (ป้องกันไม่ให้ทำงานใน case ถัดไป)
4. `default` จะทำงานหากไม่มี `case` ใดที่ตรงกับค่าของตัวแปรเลย

## goto
คำสั่ง `goto` ช่วยให้คุณกระโดดข้ามไปยังส่วนอื่นของโค้ดได้โดยตรง แม้ว่ามันจะทำงานได้จริง แต่ **ไม่แนะนำให้ใช้งาน** เพราะจะทำให้โปรแกรมเข้าใจยากและดูแลรักษาลำบาก

```cpp
#include <iostream>

int main() {
    int x = 5;

    if (x == 5)
        goto label;

    std::cout << "This line will be skipped." << std::endl;

label:
    std::cout << "Jumped to label!" << std::endl;

    return 0;
}
```
**คำอธิบาย:**
- คีย์เวิร์ด `goto` บอกให้โปรแกรมกระโดดไปยังตำแหน่งที่ระบุไว้ด้วยป้ายชื่อหรือ `label` (ตัวอย่างเช่น `label:`)
- เมื่อโปรแกรมทำงานมาถึง `goto label;` มันจะข้ามคำสั่งที่อยู่ระหว่างนั้นทั้งหมด และไปเริ่มทำงานต่อที่ตำแหน่งของ label ทันที
