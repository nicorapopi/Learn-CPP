# บทนำสู่ภาษา C++ (Introduction to C++)

C++ เป็นภาษาคอมพิวเตอร์ที่เน้นประสิทธิภาพสูงและเป็นภาษาเอนกประสงค์ (General-purpose) พัฒนาขึ้นโดย **Bjarne Stroustrup** ที่ Bell Labs เริ่มต้นในปี 1979 โดย C++ เป็นส่วนต่อขยายของภาษา C ที่มีการเพิ่มคุณสมบัติต่าง ๆ เช่น คลาส (Classes), ออบเจกต์ (Objects) และการจัดการข้อผิดพลาด (Exceptions)

---

## แนวคิดพื้นฐานของการเขียนโปรแกรม C++
นี่คือส่วนประกอบสำคัญและแนวคิดเบื้องต้นในการเขียนโปรแกรมด้วยภาษา C++:

### 1. การนำเข้าไลบรารี (Including Libraries)
ในภาษา C++ เราจะใช้คำสั่ง `#include` เพื่อนำเข้าไลบรารีหรือไฟล์หัวข้อ (Header files) เข้ามาในโปรแกรม ตัวอย่างเช่น การนำเข้าไลบรารีสำหรับอินพุต/เอาต์พุตมาตรฐาน (Standard Input/Output) เราจะเขียนว่า:

```cpp
#include <iostream>
```

### 2. ฟังก์ชันหลัก (Main Function)
จุดเริ่มต้นของการทำงานของโปรแกรม C++ คือฟังก์ชัน `main` โดยทุกโปรแกรมใน C++ จะต้องมีฟังก์ชันนี้เสมอ:

```cpp
int main() {
    // โค้ดของคุณจะเขียนที่นี่
    return 0;
}
```

### 3. อินพุตและเอาต์พุต (Input/Output)
สำหรับการรับข้อมูล (Input) และการแสดงผล (Output) ในภาษา C++ เราจะใช้ออบเจกต์สำเร็จรูปคือ `std::cin` สำหรับรับข้อมูล และ `std::cout` สำหรับแสดงข้อมูล ซึ่งอยู่ในไลบรารี `iostream`

**ตัวอย่าง: การรับค่าเลขจำนวนเต็มและแสดงผล**

```cpp
#include <iostream>

int main() {
    int number;
    std::cout << "Enter an integer: "; // แสดงข้อความ
    std::cin >> number;               // รับค่าจากผู้ใช้
    std::cout << "You entered: " << number << '\n'; // แสดงค่าที่รับมา
    return 0;
}
```

## ตัวแปรและประเภทข้อมูล (Variables and Data Types)
ภาษา C++ มีประเภทข้อมูลพื้นฐานสำหรับแทนค่าตัวเลขจำนวนเต็ม, ทศนิยม และตัวอักษร ดังนี้:

- **int**: เก็บค่าเลขจำนวนเต็ม
- **float**: เก็บค่าเลขทศนิยมแบบความละเอียดปกติ (Single-precision)
- **double**: เก็บค่าเลขทศนิยมแบบความละเอียดสูง (Double-precision)
- **char**: เก็บตัวอักษรเพียงตัวเดียว
- **bool**: เก็บค่าความจริง (จริง/เท็จ)

ตัวแปรจะต้องถูกประกาศประเภทข้อมูลก่อนใช้งานเสมอ:

```cpp
int x;
float y;
double z;
char c;
bool b;
```

## โครงสร้างการควบคุม (Control Structures)
C++ มีคำสั่งสำหรับควบคุมทิศทางของโปรแกรม ทั้งการเลือกเงื่อนไขและการวนซ้ำ เช่น `if`, `else`, `while`, `for` และ `switch`

### คำสั่งเงื่อนไข (If-Else)
```cpp
if (condition) {
    // โค้ดที่จะทำงานหากเงื่อนไขเป็นจริง
} else {
    // โค้ดที่จะทำงานหากเงื่อนไขเป็นเท็จ
}
```

### การวนซ้ำ (While Loop)
```cpp
while (condition) {
    // ทำงานซ้ำในขณะที่เงื่อนไขยังเป็นจริง
}
```

### การวนซ้ำ (For Loop)
```cpp
for (initialization; condition; update) {
    // มักใช้เมื่อทราบจำนวนรอบที่แน่นอน
}
```

### คำสั่งเลือก (Switch Statement)
```cpp
switch (variable) {
    case value1:
        // ทำงานหากตัวแปรตรงกับ value1
        break;
    case value2:
        // ทำงานหากตัวแปรตรงกับ value2
        break;
    default:
        // ทำงานหากไม่ตรงกับเงื่อนไขใดเลย
}
```

## ฟังก์ชัน (Functions)
ฟังก์ชันคือกลุ่มของโค้ดที่สามารถนำมาใช้ซ้ำได้ โดยเราสามารถเรียกใช้พร้อมส่งค่าพารามิเตอร์เพื่อให้ทำงานเฉพาะอย่าง ฟังก์ชันประกอบด้วย ประเภทการคืนค่า, ชื่อฟังก์ชัน, รายการพารามิเตอร์ และส่วนของโค้ด (Body)

```cpp
ReturnType functionName(ParameterType1 parameter1, ParameterType2 parameter2) {
    // ส่วนการทำงานของฟังก์ชัน
    return returnValue;
}
```

**ตัวอย่าง: ฟังก์ชันสำหรับบวกเลขจำนวนเต็มสองจำนวน**

```cpp
#include <iostream>

int add(int a, int b) {
    return a + b;
}

int main() {
    int result = add(3, 4);
    std::cout << "3 + 4 = " << result << '\n';
    return 0;
}
```

## บทสรุป
บทนำพื้นฐานนี้จะช่วยให้คุณมีพื้นฐานที่ดีสำหรับการเรียนรู้ในระดับที่สูงขึ้น โดยหัวข้อที่คุณควรศึกษาเพิ่มเติมมีดังนี้:
- คลาสและออบเจกต์ (Classes and Objects)
- การสืบทอดและการพหุสัณฐาน (Inheritance and Polymorphism)
- เทมเพลต (Templates)
- Standard Template Library (STL)

หัวข้อเหล่านี้จะช่วยให้คุณเข้าใจภาษา C++ ได้ลึกซึ้งยิ่งขึ้นและสามารถเขียนโปรแกรมที่ซับซ้อนขึ้นได้
